# OP_PUSHBYTES_1
:::info
**Opcode number:** 1  
**Byte representation:**  `0x01`  
**Short Description:** Push the next byte onto the stack. 
:::

The `OP_PUSHBYTES_1` opcode will push the following byte onto the stack.

## Example
```shell
# Script
OP_PUSHBYTES_1 a4

# Raw script
01a4

# Final stack
a4
```
